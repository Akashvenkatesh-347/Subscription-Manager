# Subscription-Manager

A Java console application to manage subscriptions.

## Features

- Add Subscription
- View Subscriptions
- Search Subscription
- Delete Subscription
- Save To File
- Load From File
- Input Validation
- Exception Handling

## Technologies Used

- Java 25
- Collections Framework
- File Handling
- Git
- GitHub

## Future Roadmap

- PostgreSQL Integration
- Spring Boot REST API
