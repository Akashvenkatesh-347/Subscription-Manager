# Subscription-Manager

A Java console application to manage subscriptions.

## Features

- Add Subscription
- View Subscriptions
- Search Subscription
- Delete Subscription
- Save To File
- Load From File
- Input Validation
- Exception Handling


## Project Structure
Subscription Manager
│
├── model
│     └── Subscription.java
│
├── service
│     └── SubscriptionManager.java
│
├── storage
│     └── File Storage (Current)
│
└── Main.java
